Über Herme.li

Shortlink Web-App für die Herrmann AG

Herme.li ist eine einfache Web-App um Kurzlinks für verschiedenste URLs zu erstellen. Die Seite soll es auf eine einfache Art möglich machen, firmenintern Links für alle möglichen Anwendungsfälle erstellen zu können. Egal ob für Drucksachen, Social-Media oder interne Anwendungen.

Features:

  • Erstellen und verwalten von Kurzlinks
  • Custom URLs
  • Benutzeranmeldung
  • Dashboard mit Linkübersicht
  • Statistiken zu Kurzlinks

Verwendung der Web-App ist nur für Angestellte der Herrmann AG per Login möglich.

Webseiten der Herrmann AG:

  • Herrmann-Druck.ch
  • Wochen-Zeitung.ch
  • Emmentalshop.ch
  • Verlag-Herrmann.ch
  • Kleberexperte.ch

Herme.li Technologien

Herme.li ist eine Web-App basiert auf der Open-Source App Pygmy.

Techstack

  • Python 3, JavaScript, jQuery, HTML, CSS
  • RestAPI: Flask
  • Frontend: Django
  • DB: SQLite
  • DevOps: Docker
